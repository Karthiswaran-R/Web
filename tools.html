Setting Up Linux

Follow these steps to install Linux using VirtualBox:

Prerequisites:

  • VirtualBox installed on your system.
  • Ubuntu ISO file (download it from Ubuntu's official website).

Steps to Install Ubuntu in VirtualBox:

  1. Download and Install VirtualBox:
    • Download VirtualBox from the official website.
    • Run the installer and follow the installation instructions.
  2. Create a New Virtual Machine:
    • Open VirtualBox and click "New".
    • Name it (e.g., "Ubuntu"), select Linux as type, and Ubuntu (64-bit) as version.
