Setting Up Linux

Follow these steps to install Linux using VirtualBox:

Prerequisites:

  • VirtualBox installed on your system.
  • Ubuntu ISO file (download it from Ubuntu's official website).

Steps to Install Ubuntu in VirtualBox:

  1. Download and Install VirtualBox:
    • Download VirtualBox from the official website.
    • Run the installer and follow the installation instructions.
  2. Create a New Virtual Machine:
    • Open VirtualBox and click "New".
    • Name it (e.g., "Ubuntu"), select Linux as type, and Ubuntu (64-bit) as version.
  3. Set Memory (RAM):
    • Assign 2048 MB (2 GB) or more based on system resources.
  4. Create a Virtual Hard Disk:
    • Select "Create a virtual hard disk now".
    • Choose "VDI" and "Dynamically allocated".
    • Set the size (20GB recommended).
  5. Configure the VM Settings:
    • Under "Storage", attach the Ubuntu ISO file.
  6. Start the Virtual Machine & Install Ubuntu:
    • Follow the installation steps (language, keyboard, user setup, etc.).
  7. Complete Installation:
    • Restart the VM and remove the installation disk.
    • Log in and start using Ubuntu!

Installing Synopsys Tools

Contact your university or organization to get access to Synopsys tools and follow the provided installation instructions.

Install Icarus Verilog, GTKWave, and GVim

1. Installing Icarus Verilog

  1. Open your terminal.
  2. Run: sudo apt-get install iverilog
  3. Verify installation with: iverilog -v

2. Installing GTKWave

  1. Open your terminal.
  2. Run: sudo apt-get install gtkwave
  3. Check installation with: gtkwave

3. Installing GVim

  1. Open your terminal.
  2. Run: sudo apt-get install vim-gtk
  3. Launch with: gvim
